Bot Commands

Complete list of commands for the Discord Clan Management Bot

Below you'll find a comprehensive list of all commands available in our Discord Clan Management Bot. Commands are organized by category for easy reference.

Character Commands

  • /character create - Create a new character
  • /character list - List all your characters
  • /character show_all - Show all characters for all members
  • /character find - Find a character by name
  • /character remove - Remove a character
  • /character update - Update a character's details

Clan Commands

  • /clan list - List all clans
  • /clan add - Add a new clan (admin)

Dice Commands

  • /dice roll - Roll a dice (1-100 or 1-1000)
  • /dice create_event - Create new dice event (admin)
  • /dice end_event - End current dice event (admin)
  • /dice reset_rolls - Reset all rolls (admin)

Borrow Commands

  • /borrow add - Add a borrowed item
  • /borrow list - List borrowed items
  • /borrow remove - Remove a borrowed item

Admin Commands

  • /user character_create - Create a character for another user
  • /user character_list - List all characters for a user
  • /user character_remove - Remove a character for a user

Enemy Commands

  • /enemy add - Add an enemy player (admin)
  • /enemy remove - Remove an enemy player by name (admin)
  • /enemy search - Search for enemy players
  • /enemy show_all - Show all enemy players sorted by clan
  • /enemy upload - Upload a text file with enemy players (format: name,clan). Each line should contain the player's name followed by their clan name, separated by a comma.
  • /enemy export - Export enemy players to a text file (admin)

Other Commands

  • /help - Show this help message
  • /ping - Check bot response time
  • /info about - Show bot information
